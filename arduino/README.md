# cevgf-19
COVID19 Emergency Ventilator General Firmware (CEVGF-19)

Versão de Teste para Projeto Victor/Nikolai @ 5h 15/04/2020

Comandos na serial.
Relação Ti:Te: 

't11' para Ti:Te = 1:1 
't12' para Ti:Te = 1:2
't13' para Ti:Te = 1:3 

PWM:

'vN' onde N é a porcentagem do PWM (0 a 100%). ex: 'v50' para 50% do PWM.
